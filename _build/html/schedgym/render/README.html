VMAgent UI

Requirements

  • Python 3.7+

  • Dash 1.21.0

  • Dash DAQ 0.5.0

  • Dash Bootstrap Components 0.13.0

  • Dash Bootstrap Templates 0.1.1

  • Numpy

  • Pandas

  • Plotly 5.3.0

Install requirements and start dev server

pip install -r requirements.txt
python run.py

Deploy to production

Install gunicorn, and run the following command.

gunicorn dashboard.wsgi
